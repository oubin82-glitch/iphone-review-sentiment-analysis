# 基于 LDA 模型的手机评论情感分析
**Sentiment Analysis of iPhone 15 Reviews via LDA Topic Modeling**

> 爬取京东平台 10 万余条 iPhone 15 用户评论，综合运用情感分析与 LDA 主题建模，挖掘用户核心关注点，为产品改进提供数据支撑。

---

## 项目概述

电商平台的用户评论蕴含大量真实的消费反馈。本项目以 **iPhone 15** 为研究对象，从京东商城采集 10 万余条评论数据，通过完整的 NLP 分析流程，识别用户正负情感分布及核心关注主题。

**核心发现：**
- 负面评论集中于**信号稳定性**与**售后服务**
- 正面评论主要围绕**拍照体验**、**屏幕显示**与**系统流畅性**
- LDA 建模识别出 3 个正面主题、3 个负面主题

## 技术栈

| 模块 | 工具 / 库 |
|------|-----------|
| 数据采集 | Python `requests` + `lxml`（京东评论 API） |
| 数据处理 | `pandas`、`numpy`、`re` |
| 中文分词 | `jieba`（含词性标注） |
| 情感分析 | 清华情感词典（正负词典匹配 + 否定词修正）、`pysenti` |
| 主题建模 | `gensim` LDA，余弦相似度寻优主题数 |
| 可视化 | `matplotlib`、`wordcloud`、`pyLDAvis` |

## 分析流程

```
数据采集 → 数据清洗 → 分词 & 词性过滤 → 去停用词
    ↓
情感词匹配（清华词典）→ 否定词修正 → 情感得分计算 → 正/负分类
    ↓
LDA 主题建模（余弦相似度寻优）→ 主题解读 → 词云 & pyLDAvis 可视化
```

### 关键步骤说明

**1. 数据采集**
通过分析京东商品评论接口，采集 iPhone 15 多个 SKU（颜色/容量版本）的好评、中评、差评共计 **101,000 条**，字段包括：用户 ID、昵称、评分、评论正文、评论时间等。

**2. 数据清洗**
- 去除重复评论（`drop_duplicates`）
- 正则过滤数字、英文字母及高频无意义词（"京东"、"苹果"、"手机"等）
- `jieba` 词性标注，仅保留名词（`n`类）

**3. 情感分析**
- 使用清华大学情感词典构建正负情感词表
- 对每条评论按词顺序进行情感值累加，检测否定词（"不"、"没有"等）并取反
- 汇总得出每条评论情感极性（积极 / 消极 / 中性）

**4. LDA 主题建模**
- 对正面、负面评论分别建模
- 使用余弦相似度法确定最优主题数（K=3）
- `pyLDAvis` 生成交互式主题可视化

## 主要结论

| 情感类型 | 主题 | 核心关键词 |
|----------|------|-----------|
| 正面 | 拍照 & 外观 | 相机、镜头、颜色、设计 |
| 正面 | 性能 & 系统 | 流畅、速度、系统、运行 |
| 正面 | 续航 & 配件 | 电池、充电、耳机、配件 |
| 负面 | 信号问题 | 信号、网络、基站、4G |
| 负面 | 售后 & 物流 | 退货、客服、发货、包装 |
| 负面 | 价格 & 性价比 | 价格、降价、贵、性价比 |
